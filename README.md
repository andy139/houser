# StockHome

StockHome, a RoofStock clone, is an online marketplace that allows users to browse, purchase, and bid on properties.

## Technologies

* Ruby on Rails
* Postgres
* React & Redux
* Google Maps API

## Features
* User and Session Authentication
* Filter Search
* Add to cart
* Google Maps
